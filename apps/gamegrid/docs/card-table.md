# Card Table

Card Table is a multi-mode hub at `/play/card-table`.

## Disclaimer
No real money. Soft currency only. For entertainment.

## Modes
- Blackjack
- Higher or Lower
- 31
- Solitaire (Klondike)
- Forehead Poker (Indian Poker)
- 5 Card Draw
- Texas Hold'em Poker

## Controls
- Touch-first large tap targets, mouse supported
- `Esc` pause and `M` mute are handled by the portal shell
- Back button returns to mode select (with mid-round confirmation)

## Bankroll and Stats
- Shared soft-currency bankroll (default enabled)
- Per-mode stats: plays, wins/losses/pushes, best streak, bankroll delta
- Local-only persistence in browser storage
