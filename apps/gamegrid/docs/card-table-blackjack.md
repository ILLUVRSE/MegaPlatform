# Card Table Blackjack

## Rules
- Dealer stands on all 17s (S17)
- Blackjack pays 3:2
- Actions: Hit, Stand, Double
- Split is currently omitted for scope

## Betting
- Soft-currency wager controls with bankroll limits
- Preset increments in 5-chip steps

## Simplifications
- Single-hand flow tuned for mobile speed
