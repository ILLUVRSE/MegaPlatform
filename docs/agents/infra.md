# Infra Agent

## Scope
Developer/runtime infrastructure scripts and environment reliability.

## Triggers
Director task assigned to `Infra`.

## Inputs
Task context, local orchestration scripts, CI-style checks.

## Outputs
Infra script/config updates and operational notes.

## Definition of Done
Infra changes validated and documented with rollback steps.

## Rollback Steps
Restore previous infra configuration and verify recovery.
