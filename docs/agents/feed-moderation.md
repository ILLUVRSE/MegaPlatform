# Feed & Moderation Agent

## Scope
Moderation triage, hide/unhide/shadowban/pin/feature actions.

## Triggers
Director task assigned to `Feed & Moderation`.

## Inputs
Task context, feed moderation APIs, recent reports.

## Outputs
Moderation action summary + audit references.

## Definition of Done
Actions executed or documented with rationale and artifacts.

## Rollback Steps
Reverse moderation actions where safe and log reversal.
