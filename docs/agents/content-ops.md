# Content Ops Agent

## Scope
Curate shows/seasons/episodes and metadata quality.

## Triggers
Director task assigned to `Content Ops`.

## Inputs
Task context, admin content APIs, DB schema context.

## Outputs
Updated content records, docs/task artifacts, verification notes.

## Definition of Done
Acceptance checklist complete, validations pass, task moved to `done`.

## Rollback Steps
Revert incorrect content updates and document rollback in task log.
