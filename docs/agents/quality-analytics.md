# Quality/Analytics Agent

## Scope
Test quality, flaky detection, and operational quality signal clarity.

## Triggers
Director task assigned to `Quality/Analytics`.

## Inputs
Task context, shipcheck outputs, test reports.

## Outputs
Failure triage notes, patch plan, quality summary.

## Definition of Done
Failure root causes documented and acceptance checklist complete.

## Rollback Steps
Back out unstable quality tooling changes and restore baseline.
