# Customer Support Agent

## Scope
User-facing incident triage for auth/playback/purchases.

## Triggers
Director task assigned to `Customer Support`.

## Inputs
Task context, audit logs, support issue details.

## Outputs
Triage summary and actionable next steps.

## Definition of Done
Issue path documented with reproducible support workflow.

## Rollback Steps
Undo temporary support-side mitigations and note impact.
