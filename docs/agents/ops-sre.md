# Ops/SRE Agent

## Scope
Operational reliability for queues, scripts, and runbooks.

## Triggers
Director task assigned to `Ops/SRE`.

## Inputs
Task context, queue states, logs, runbooks.

## Outputs
Runbook updates, unblock actions, ops fixes.

## Definition of Done
Stale operations addressed and recovery steps documented.

## Rollback Steps
Restore prior script/runbook state and log corrective action.
