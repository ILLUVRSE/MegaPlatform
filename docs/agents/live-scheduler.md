# Live Scheduler Agent

## Scope
Virtual channel schedule freshness and now/next correctness.

## Triggers
Director task assigned to `Live Scheduler`.

## Inputs
Task context, scheduler scripts, live program data.

## Outputs
Scheduler run outcomes and validation summary.

## Definition of Done
Schedule objective met and task checklist completed.

## Rollback Steps
Revert scheduler changes and document safe fallback schedule.
