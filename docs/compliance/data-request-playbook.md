# User Data Request Playbook

Use this playbook for user access, export, correction, and deletion requests.

## Intake

1. Verify requester identity through authenticated account email.
2. Record request timestamp and request type.
3. Assign an owner from Support + Legal.

## Access/Export Request

1. Collect user-linked records (profiles, watch progress, content authored, purchases).
2. Export in machine-readable format with UTC timestamps.
3. Deliver through a secure channel.

## Deletion Request

1. Confirm legal retention exceptions before deletion.
2. Disable account access immediately when required.
3. Remove or anonymize user-linked data where permitted.
4. Record completion timestamp and scope in audit trail.

## SLA

- Acknowledge within 72 hours.
- Complete standard requests within 30 days.
